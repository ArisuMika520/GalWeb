本篇文章是阐述本站的构成，来源以及背后的一些运作

介绍本站

网站的自我介绍（雾）

LyCorisGAL 是一个 Gal 引导资源站，致力于为大家提供 Galgame 相关的资源和信息。提供Galgame简短信息、Galgame下载等服务。
本网站目前没有任何盈利！主要是为Galgame社区和玩家们提供方便。

本站由ArisuMika独自维护运营。采用的是hexo框架的静态网页。所以暂且不支持用户登录、Gal资源收藏、上传资源等功能。

网站的一些问题

访问网络问题

LyCorisGAL是采用hexo框架的静态网页，服务器在国外，虽然采用了加速技术，但是在一些时候国内访问依旧会有不流畅、卡顿甚至无法访问的问题。请自行解决网络问题，懂得应该都懂。要么你过一段时间再访问。

存储与传输问题

存储

因为本站由本人独自一人维护，而且本人是大一（2024）穷学生，并不能提供非常稳定且快速的对象存储储存。所以本站资源存储方式非常多样，对于失效或者没有的资源，本站表示抱歉，请自行前往友站进行搜索与下载。

存储方式

本站目前采用的存储有：

  1. 少量的CloudFlare R2对象存储
  2. 较少的OneDrive存储
  3. 123云盘存储
  4. 中国移动云盘

为什么使用123云盘：
1. 123云盘并不限速
2. 123成本较低，提供大量容量
3. 123支持直链接下载和无客户端下载流量（注意：由于这种流量也需要额外的费用开销，所以后来者可能没有免客户端流量可以使用，请自行跟据指引下安装客户端进行资源下载）

为什么要用中国移动云盘！：
1. 中国移动云盘并不限速
2. 中国移动云盘成本较低，提供大量容量
3. 中国移动云盘支持直链下载
注意：中国移动云盘可能某天不支持直链了，请自行使用客户端下载。

少量的CloudFlare R2对象存储是多少：不超过1TB，根据个人经济情况进行动态调整。现阶段不会超过100G
CloudFlare R2和OneDrive的下载慢怎么办：下载服务器在国外，请自行解决网络问题

其他问题

每次进网站都要看加载动画很烦怎么办？（？不是哥们）

加载的时候点击一下屏幕就可以了，但是可能会出现未加载完全的情况，加载动画就是等加载完全再进入页面设置的。

如何获取本站最新网址，防止失联？

加入本站tg用户群获取最新网址。旧网址在失效前2个月会自行跳转到新网址，请自行及时收藏。

如何向本站捐赠/赞助？

目前没有考虑捐助渠道，后面会考虑。

可以分享和宣传本站吗？

可以，而且欢迎友链添加！

本站资源的来源

本站资源来自互联网以及部分个人自购，资源完全免费。若看到有人私自贩卖资源与本站链接，请自行举报，谢谢。

为什么资源有点少？

本站处于刚起步阶段，资源会慢慢补充。而且由于个人经济问题，难以承担巨大的存储费用。

网站图片/视频无法显示？

网络问题自行解决。或者可能你浏览器太垃圾了，自行升级或者改用当前主流浏览器。

自建网盘下载下来的是几十KB大小的压缩包？

请不要使用打包下载，一个一个右键点击下载，而且速度请不要过快，等待跳出下载页面之后再点击下一个文件的下载。

下载的资源解压失败报错？

大概率是因为网络问题，在下载的过程中环境不佳导致的。请自行解决网络问题重新下载。
也有可能是其他原因，若多次重新下载后依旧报错，检查文件格式与对应的解压步骤规范、解压工具。
在此之外，依旧报错，请联系本人（xyxx1907653220@outlook.com）或者加入telegram群组联系本人。

访问自建网盘网络错误？无法访问？

自建网盘也是搭建在国外的服务器上，自行解决网络问题。

能不能攻击测压网站？

请不要攻击！！！你的一次攻击可能导致本站全部崩溃，本人没有财力与精力进行多余的防御和崩溃再次组织能力。

能不能注册自建网盘？

不行，自建网盘仅私用。

为什么叫LyCoris?

因为好听，取自彼岸花
